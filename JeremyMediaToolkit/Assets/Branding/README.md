# Branding assets

- `jr-glow-source.png` is an unchanged project copy of `Glow Logo square No Text.png`, supplied by Jeremy Running Photography.
- `JeremyMediaToolkit.ico` is a deterministic Windows icon derived from that source and contains 16, 24, 32, 48, 64, 128, and 256 pixel layers.

The source logo remains the visual reference for in-application branding. Do not regenerate or restyle the JR mark when producing additional application assets.
